# Piler

## Usage

```
# 1, 2 are project id
$ piler --repository=meganemura/piler --from=1 --to=2
Piler moves cards from 'my-project-1' to 'my-project-2'
* my-project-1/project-1-column-a
* my-project-1/project-1-column-b
* my-project-1/project-1-column-b
Finished.
```

## Installation

    $ gem install piler
