# Piler

## Usage

```
# Specify project numbers (`github.com/owner/repo/projects/{project_number}`)
$ piler --repository=meganemura/piler --from=1 --to=2
Piler moves cards from 'FROM' to 'TO'
  Copy from-column-1
    * card2
    * card1
  Copy from-column-2
    * card-b
    * card-a
```

## Installation

    $ gem install piler
